## Backend Deployment

To deploy this project run following commands on Backend folder

```bash
  npm install
```

```bash
  npm run dev
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`mongoURL`

`normalKey`

`refreshkey`

`port`

`crypterKey`

`redisURL`
